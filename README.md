# ARIA config

The roles and properties from [WAI-ARIA](http://w3c.github.io/aria/aria/aria.html) specification in a JSON format.

The major and minor versioning numbers refers to the aria specification version.

These files are compiled manually from the latest editors draft as W3C does not appear to supply this information in an up to date format.

* [`roles.json`](roles.json) - ARIA roles
* [`attributes.json`](attributes.json) - ARIA attributes
